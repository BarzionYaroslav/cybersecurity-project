# Project for Cybersecurity (File Storage)

## Description

This is the cybersecurity project for a simple file storage made with FastAPI. It has a simple way to add and change users, as well as adding files from the users to the database.

## Technology

The project uses FastAPI for the actual input and Docker for the file storage. Fernet is used to encode the secret files

## Launch

- *Clone* the repository on your machine

- Execute `cp .env.example .env` and edit the .env file or create one entirely new

- Fill in the info in the .end

- Execute `docker-compose up -d`

## API

API can be used through the `/docs` page (usually `http://127.0.0.1:8000/docs/`)

## Placement of tasks

Tasks can be found in the folder /tasks
